# 新世纪机器人社图形界面API
## 构建说明
* 1 本项目大量使用c++17特性 故必须使用  `GNU Tools for Arm Embedded Processors 8-2018-q4-major` 以上的版本编译.早期版本必然无法通过编译
* 2 本项目使用了 `pros库文件.` 但并未使用`pros`的开发环境.开发环境用户可以根据喜好自行定制.
* 3 提示:直接使用`pros IDE`的用户是无法通过编译的.需要自行搭建开发环境.
